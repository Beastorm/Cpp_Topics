### 1. Introduction to Static Members
**What Are Static Members?**
Static members belong to the class itself, not to any particular object. They are shared among all instances of the class.

``` 
┌─────────────────────────────────────────────────────────────────┐
│                    STATIC vs INSTANCE MEMBERS                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  INSTANCE MEMBERS (Non-Static):                                 │
│  ──────────────────────────────                                 │
│  • Each object has its OWN copy                                 │
│  • Created when object is created                               │
│  • Destroyed when object is destroyed                           │
│  • Accessed through object: obj.member                          │
│                                                                 │
│  STATIC MEMBERS:                                                │
│  ───────────────                                                │
│  • ONE copy shared by ALL objects                               │
│  • Exists even if NO objects exist                              │
│  • Created at program start, destroyed at program end           │
│  • Accessed through class: ClassName::member                    │
│  • Can also access through object: obj.member (not recommended) │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Visual Representation**

``` 
═══════════════════════════════════════════════════════════════════
                    MEMORY LAYOUT
═══════════════════════════════════════════════════════════════════

    class Counter {
        static int totalCount;    // Static: shared
        int instanceId;           // Non-static: per object
    };

    STATIC DATA SEGMENT                    HEAP/STACK (Objects)
    ┌────────────────────┐                ┌─────────────────┐
    │                    │                │   Counter obj1  │
    │  totalCount: 3     │◄───────────────┤   instanceId: 1 │
    │                    │                └─────────────────┘
    │  (shared by all)   │                ┌─────────────────┐
    │                    │◄───────────────┤   Counter obj2  │
    └────────────────────┘                │   instanceId: 2 │
             ▲                            └─────────────────┘
             │                            ┌─────────────────┐
             └────────────────────────────┤   Counter obj3  │
                                          │   instanceId: 3 │
                                          └─────────────────┘

    • ONE totalCount exists in static segment
    • EACH object has its own instanceId
    • All objects SHARE the same totalCount

═══════════════════════════════════════════════════════════════════  
```

**Basic Example**

``` cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
private:
    string name;
    int id;
    
    static int totalStudents;    // Static data member (declaration)

public:
    Student(const string& n) : name(n) {
        id = ++totalStudents;    // Increment shared counter
        cout << "Student " << name << " created with ID " << id << endl;
    }
    
    ~Student() {
        totalStudents--;
        cout << "Student " << name << " destroyed" << endl;
    }
    
    // Static member function
    static int getTotalStudents() {
        return totalStudents;
    }
    
    // Instance member function
    void display() const {
        cout << "ID: " << id << ", Name: " << name << endl;
    }
};

// Static member definition (required outside class)
int Student::totalStudents = 0;

int main() {
    cout << "Total students initially: " << Student::getTotalStudents() << endl;
    
    {
        Student s1("Alice");
        Student s2("Bob");
        Student s3("Charlie");
        
        cout << "\nTotal students now: " << Student::getTotalStudents() << endl;
        
        s1.display();
        s2.display();
        s3.display();
        
        cout << "\nLeaving scope..." << endl;
    }
    
    cout << "\nTotal students after scope: " << Student::getTotalStudents() << endl;
    
    return 0;
}
```

**Output:**

```text

Total students initially: 0
Student Alice created with ID 1
Student Bob created with ID 2
Student Charlie created with ID 3

Total students now: 3
ID: 1, Name: Alice
ID: 2, Name: Bob
ID: 3, Name: Charlie

Leaving scope...
Student Charlie destroyed
Student Bob destroyed
Student Alice destroyed

Total students after scope: 0
```
